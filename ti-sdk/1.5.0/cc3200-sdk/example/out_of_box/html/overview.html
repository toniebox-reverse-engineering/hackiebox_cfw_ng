The SimpleLink CC3200 product is a wireless microcontroller (MCU) with:
  • 80 MHz ARM® Cortex™-M4 processor
  • On-chip memory
  • Peripherals
  • Wi-Fi network processor

The CC3200 LaunchPad is designed to be used as a standalone development platform for application development using the CC3200 device. It can be also used in conjunction with compatible BoosterPack plug-in modules to enhance the peripherals available in the system. It features on-board emulation using FTDI device and has an array of sensors to deliver an easy out-of-the-box experience. This board can be directly connected to the PC using software development platforms including CCS and IAR.

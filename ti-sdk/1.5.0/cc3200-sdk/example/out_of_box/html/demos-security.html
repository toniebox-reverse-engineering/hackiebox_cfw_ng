Safety and Security

Toggle switch ON to turn on the system. Pick up and shake the LaunchPad to see the Alarm on the page go off.

On
Off

The SimpleLink CC3200 has powerful crypto engines that enable your application to safely connect to the cloud.

It supports all the Wi-Fi security protocols such as :

  • WPA2, WPS2, 802.1x, EAP Fast, EAP PEAP, EAP TLS.

It also supports internet security such as:

  • SSL3.0, TLS 1.2, X.509, DES3, AES256, MD5, SHA2, RSA, ECC

By simply shaking the LaunchPad, it triggers the accelerometer and sets off the alarm (display and LEDs).

Did you know?

CC3200 has fast and secure performance because it has on-chip hardware accelerators (Crypto engine) to enable connections with 256 bit encryption.

Customer Application - easy development for applications such as: home automation and security, connected home, smart energy, industrial automation, and many more
Simplelink driver – Common between CC3100 and CC3200
SPI driver – 4-wire bidirectional communications interface that converts data between parallel and serial. Interface between CM4 and NWP
Host Interface – Wide range of VIO (1.8-3.6V), SPI, UART (3100 only) driver
Network apps – ARP, ICMP, DHCP, DNS mDNS, http server
SimpleLink connection manager – WLAN profiles and security management
TCP/IP stack – IPv4
WLAN MAC & PHY – 2.4GHz radio + physical layer
OS – Can work with Free RTOS or TI RTOS, also no OS
Peripheral Drivers – camera, audio

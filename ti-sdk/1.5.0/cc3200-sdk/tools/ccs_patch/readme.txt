
CCS Target Configuration Install
---------------------------------
1. The Target Configuration File (.ccxml) can be copied to “C:\Users\<username>\ti\CCSTargetConfigurations” or imported from the CCS IDE.

Note-
* CC3200.ccxml: used for FTDI based CC3200 device (Launchpad)
* CC3200_xds100v3.ccxml: if device is connected through XDS100v3
